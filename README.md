TO-DO list Application
A responsive To-Do List application built using React.js with task management features, voice input using Web Speech API, and reminder notifications using Capacitor. The app is fully deployed and accessible live.

Features :
Add, complete, and delete tasks
Voice input using Web Speech API
Reminder notifications with date/time
Responsive UI using Bootstrap
Live deployment

Tech Stack :
React.js
JavaScript
Bootstrap
Capacitor
Web Speech API
